## 财务指标体系

### 盈利能力指标
- 毛利率 = (收入 - 成本) / 收入
- 净利率 = 净利润 / 收入
- ROE = 净利润 / 股东权益

### 运营效率指标
- 存货周转率 = 销售成本 / 平均存货
- 应收账款周转率 = 收入 / 平均应收账款
- 总资产周转率 = 收入 / 总资产

### 现金流指标
- 自由现金流 = 经营现金流 - 资本支出
- 现金流覆盖率 = 经营现金流 / 负债
