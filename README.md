# claude-skills
个人使用的 Claude Skill 合集

## 介绍
- ceo: 首席执行官专家
- cfo: 财务总监专家
- product-manager: 产品经理专家
- ui-ux-designer: UI/UX 设计师专家
- tech-lead: 技术负责人专家
- marketing-operations: 市场运营专家
- legal-advisor: 法务顾问专家
- cio: 首席信息官专家
- cto: 首席技术官专家
- kuikly-develop: Kuikly 开发技能
- miniapp-develop: 小程序开发技能

## 使用
将此仓库克隆到本地，链接Skill到~/.claude/skills/对应目录。
```shell
cd ~/
git clone git@github.com:lotosbin/claude-skills.git
ln -s `pwd`/claude-skills/.claude/skills/ceo `pwd`/.claude/skills/ceo
ln -s `pwd`/claude-skills/.claude/skills/cfo `pwd`/.claude/skills/cfo
ln -s `pwd`/claude-skills/.claude/skills/product-manager `pwd`/.claude/skills/product-manager
ln -s `pwd`/claude-skills/.claude/skills/ui-ux-designer `pwd`/.claude/skills/ui-ux-designer
ln -s `pwd`/claude-skills/.claude/skills/tech-lead `pwd`/.claude/skills/tech-lead
ln -s `pwd`/claude-skills/.claude/skills/marketing-operations `pwd`/.claude/skills/marketing-operations
ln -s `pwd`/claude-skills/.claude/skills/legal-advisor `pwd`/.claude/skills/legal-advisor
ln -s `pwd`/claude-skills/.claude/skills/cio `pwd`/.claude/skills/cio
ln -s `pwd`/claude-skills/.claude/skills/cto `pwd`/.claude/skills/cto
ln -s `pwd`/claude-skills/.claude/skills/kuikly-develop `pwd`/.claude/skills/kuikly-develop
ln -s `pwd`/claude-skills/.claude/skills/miniapp-develop `pwd`/.claude/skills/miniapp-develop
ln -s `pwd`/claude-skills/.claude/skills/task-manager `pwd`/.claude/skills/task-manager
```
